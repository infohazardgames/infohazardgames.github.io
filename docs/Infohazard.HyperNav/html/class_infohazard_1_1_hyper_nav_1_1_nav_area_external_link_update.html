static

Generate external links for the given areas. Can be used at runtime to support dynamically created links.

Parameters
areasAreas to generate external links for. Regardless of this parameter, all loaded areas will be considered as destinations for external links.
updateSerializedDataWhether to update serialized data for the areas. If this is false, GC allocations will be avoided, but the data will not be saved.
keepLinksToUnloadedScenesIf true, links to areas in unloaded scenes will be kept.
maxCompletionFramesMaximum number of frames to wait for job completion. -1 for no limit.
Exceptions
OperationCanceledExceptionThrown if the job was cancelled due to data changing.
